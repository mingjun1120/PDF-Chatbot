## Run Locally

Clone the project

```bash
  git clone https://github.com/mingjun1120/PDF-Chatbot.git
```

Go to the project directory

```bash
  cd PDF-Chatbot
```

Install dependencies

```bash
pip install -r requirements.txt
```

To run the app, run the following command in the terminal

```bash
python main.py
```
